# 高斯模糊

```jsx
	background: #cccccc50; // 背景色需要 #[颜色代码]+[0-100透明度]
  -webkit-backdrop-filter: blur(20px);//其他浏览器代码
  backdrop-filter: blur(20px) contrast(.5) ;//chrome代码
	//blur 模糊 
	//contrast 对比度
```
